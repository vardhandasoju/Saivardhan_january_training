# Customer Segmentation using Unsupervised Learning

## 📌 Project Overview
This project performs customer segmentation using unsupervised machine learning techniques. The goal is to identify different groups of customers based on their purchasing behavior.

---

## 📊 Dataset
- Source: Kaggle Marketing Campaign Dataset
- Total Records: ~2240 customers
- Features: Income, Age, Spending, Purchases, Recency

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 🧠 Algorithms Used
- KMeans Clustering
- Hierarchical Clustering
- DBSCAN
- Gaussian Mixture Model (GMM)

---

## 📈 Project Workflow

1. Data Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering (RFM Model)
4. Clustering
5. Model Evaluation
6. Visualization

---

## 📊 Evaluation Metrics

- Silhouette Score : 0.24523696359524327
- Davies-Bouldin Score : 1.513219738015584

---

## 📉 Results

- Optimal clusters identified using Elbow Method
- Customer segments visualized using PCA
- Cluster-wise behavior analyzed
